# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it by emailing [INSERT CONTACT EMAIL].

- Provide a detailed description of the vulnerability.
- Include steps to reproduce the issue if possible.
- Do not disclose the vulnerability publicly until it has been addressed.

We will respond as quickly as possible and keep you updated on the progress of the fix.

## Supported Versions

| Version | Supported          |
| ------- | ----------------- |
| latest  | :white_check_mark:|

Older versions may not be supported. Please use the latest version for security updates.
